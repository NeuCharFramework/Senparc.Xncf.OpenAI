## 关于 Senparc.Xncf.XncfBuilder.Template.*.nupkg 文件

`Senparc.Xncf.XncfBuilder.Template.*.nupkg` 文件用于在新建 XNCF 项目时，提供本地的模板 Nuget 包文件，如果不存在此文件，将自动转为在线获取。
